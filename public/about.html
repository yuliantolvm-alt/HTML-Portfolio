About Me

Halo! Saya sedang berada di tahap awal perjalanan sebagai Web Developer, dengan fokus utama saat ini pada HTML. Saya mempelajari bagaimana sebuah website dibangun dari struktur dasarnya — mulai dari penggunaan elemen, layout sederhana, hingga penulisan kode yang rapi dan mudah dipahami.

Melalui proyek-proyek kecil berbasis HTML, saya berlatih membangun halaman web yang terstruktur dengan baik dan siap untuk dikembangkan lebih lanjut. Proses ini membantu saya memahami peran HTML sebagai fondasi penting sebelum melangkah ke tahap berikutnya dalam pengembangan web.

Saya sangat bersemangat untuk melanjutkan pembelajaran ke CSS untuk mempercantik tampilan, JavaScript untuk menambahkan interaksi, dan akhirnya memahami konsep full-stack web development secara bertahap. Bagi saya, belajar web development adalah proses berkelanjutan, dan setiap langkah kecil memiliki peran besar dalam perkembangan kemampuan saya.

Portofolio ini merupakan catatan awal dari perjalanan belajar saya — dimulai dari HTML, dengan komitmen untuk terus berkembang dan mengeksplorasi teknologi web selanjutnya.
